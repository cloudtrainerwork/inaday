Quickstart

  1. Start by downloading, and extracting, the required files from this location

  2. The files should resemble the following after extraction

  3. In VS Code open the folder where these files are.

  4. Expand the exercises folder and double-click the quickstart.ipynb to open the notebook file.

  5. You will probably be prompted to allow a couple of extensions to be setup.

  6. In the upper-right hand corner the Kernel needs to be selected

  7. Click on the Select Kernel button to choose the Python environment you wish to use. Tested on Python 3.10.0. Select Create Python Environment

  8. Create a Venv

  9. Select Python 3.10.0

  10. Select the requirements.txt file

  11. Let the packages required install

  12. First, create a file called .env in the exercises folder, and add the following content

    OPENAI_API_KEY=xxxxxx
    OPENAI_API_BASE=https://xxxxxxx.openai.azure.com/

  1. There are two different ways to authenticate but the code has been updated to use the Access Key authentication. The requirements file has also been updated for Python versions greater than 3.9

  2. To execute a cell click on the triangle on the left side of the cell.
